# 🌮 La Taquería Poblana - Landing Page

Bienvenido al repositorio de **La Taquería Poblana**, un proyecto de desarrollo web estático enfocado en crear una experiencia de usuario moderna, responsiva y visualmente atractiva para un restaurante tradicional mexicano. 

Este proyecto fue desarrollado cumpliendo los lineamientos de la **Norma 24** para garantizar los más altos estándares de calidad en su estructura y presentación.

## 🚀 Características Principales

* **Diseño 100% Responsivo:** Se adapta perfectamente a dispositivos móviles, tablets y pantallas de escritorio.
* **Estilo Moderno (Tailwind CSS):** Utilización de clases utilitarias para un diseño limpio sin necesidad de archivos CSS externos.
* **Tipografía Premium:** Integración de Google Fonts (*Playfair Display* para títulos y *Manrope* para textos) para una lectura fluida y elegante.
* **Iconografía Profesional:** Uso de *FontAwesome* para redes sociales y elementos de la interfaz.
* **Componentes Integrados:** * Barra de navegación fija (Sticky Navbar) con scroll suave.
    * Sección de Hero con imagen de fondo a pantalla completa.
    * Menú de platillos en formato de cuadrícula (Grid).
    * Mapa interactivo de Google Maps embebido (Iframe).

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica del sitio.
* **Tailwind CSS (v3 via CDN):** Framework principal para los estilos y el diseño responsivo.
* **Google Fonts:** Tipografía.
* **FontAwesome (v6):** Iconos vectoriales.
* **Unsplash:** Imágenes de alta calidad libres de derechos.
